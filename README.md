# Nitrogen

A C-Style programming language

## Building Nitrogen

Ensure that makefile is installed.
Run the makefile by running the `make` command.

## Generating Documentation

Make sure doxygen is installed, then run
Run `doxygen nitrogen_docs`

## Reference

### Builtin Functions

#### Print:

`print(STRING)`: Prints to stdout

**Example:**
```
print("Hello, World!");
```
